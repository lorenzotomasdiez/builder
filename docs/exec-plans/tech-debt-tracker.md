## How to Use

1. **Adding debt**: When you notice a pattern or issue, add it above
2. **Prioritizing**: Review during planning, prioritize by impact
3. **Resolving**: When fixed, move to Resolved section with resolution date
4. **Agent awareness**: Agents can read this to understand known issues

## Guidelines

- Be specific about the problem
- Include impact level (helps prioritization)
- Add date for aging tracking
- Link to exec-plans if work is planned
